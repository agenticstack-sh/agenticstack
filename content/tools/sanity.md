# Sanity

Sanity is a headless CMS with a real-time collaborative editing experience. Its custom query language (GROQ) is powerful for complex content fetches, and the schema is fully code-defined — no UI-only content modeling.

For AI agents, Sanity's generous free tier (500k API requests/month) makes it practical for prototyping content-aware agents. The real-time subscription API also enables agents that react to content changes as they happen.
