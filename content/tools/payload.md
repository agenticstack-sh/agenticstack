# Payload

Payload is an open-source, code-first CMS built on TypeScript and designed to work natively with Next.js. It supports both SQL (Postgres) and MongoDB as backends and generates type-safe APIs automatically from your schema definitions.

For AI agent workflows, Payload's config-as-code approach means content models are version-controlled and programmatically accessible. The tight Next.js integration makes it a natural choice for teams already in that ecosystem.
