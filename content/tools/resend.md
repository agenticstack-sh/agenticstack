# Resend

Resend is a developer-first email API built by the team behind React Email. Its SDK design is clean and modern — sending a transactional email is a single function call with a typed interface.

For AI agent developers, Resend is a solid choice when your agent needs to send emails (notifications, summaries, reports) as part of its workflow. The API is straightforward to wrap as a tool in any agent framework.
